# SmartStock
Sistem Manajemen Stok Barang adalah aplikasi sederhana berbasis OOP yang digunakan untuk mencatat, mengatur, dan memantau jumlah persediaan barang. Sistem ini membantu pengguna menambah data barang, memperbarui stok, dan mengetahui barang yang hampir habis sehingga pengelolaan inventaris menjadi lebih rapi, cepat, dan terkontrol.
